# Traffic Light 4 Arah Arduino

## Deskripsi
Simulasi traffic light 4 arah menggunakan Arduino Uno di Tinkercad.

## Komponen
- Arduino Uno
- 12 LED (Merah, Kuning, Hijau)
- 12 Resistor 220Ω
- Breadboard
- Kabel jumper

## Pin
Utara: 2,3,4  
Timur: 5,6,7  
Selatan: 8,9,10  
Barat: 11,12,13  

## Cara Kerja
Lampu hijau menyala bergiliran searah jarum jam: Utara → Timur → Selatan → Barat.
Setiap arah:
- Hijau 5 detik
- Kuning kedip 3x lalu 2 detik
- Kembali merah
